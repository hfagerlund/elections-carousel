Style guide

Naming conventions

Functions, Methods

Camel case starting with lowercase letter. Private method names prefixed with '_' (underscore).

Example - Private method:

function _doSomething(foo) {
    //...
}

Modules

Camel case starting with uppercase letter.

Variables

Camel case starting with lowercase letter.

Linting

Refer to .eslintrc .

Spacing

2 spaces for indentation. No whitespace at EOL.

Refer to .editconfig .
